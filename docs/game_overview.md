Rabbit Rush is a crash-style betting game inspired by classic multiplier games.

## Game Concept
- A rabbit runs through a jungle while the multiplier increases from 1.00x upward.
- The round ends when the rabbit stops (crash).
- Players may cash out at any time before the crash.

## Player Actions
- Place a bet before the round starts.
- Optionally set an auto-cashout multiplier.
- Manually cash out at any time before the crash.

## Outcomes
- If the player cashes out before the crash, the bet wins.
- If the crash occurs before cashout, the bet loses.

## Game Properties
- Stateless
- Each bet is independent
- No jackpots
- No continuation or gamble features
- Fully deterministic and replayable

## Compliance
This game follows StakeEngine crash game requirements and supports Bet Replay.
